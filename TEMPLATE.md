# Skill Template

Use this format when creating new skills:

```markdown
---
name: skill-name-in-kebab-case
description: |
  Brief description with trigger conditions. Use when: (1) specific symptom,
  (2) another trigger, (3) context clues. Helps with [problem domain].
author: Your Name
version: 1.0.0
date: YYYY-MM-DD
---

# Skill Name

## Problem
Clear description of the problem this skill addresses.

## Context / Trigger Conditions
- When should this skill be used?
- Exact error messages
- Symptoms or scenarios

## Solution
Step-by-step solution or knowledge to apply.

## Verification
How to verify the solution worked.

## Example
Concrete example of applying this skill.

## Notes
Any caveats, edge cases, or related considerations.

## References
Links to documentation or resources (optional).
```
